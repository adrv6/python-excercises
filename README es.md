# 🐍 Ejercicios de Python

Una colección de ejercicios de Python para principiantes completados a través de [Codedex](https://www.codedex.io/) y [Codewars](https://www.codewars.com/), organizados por tema. Este repositorio documenta mi proceso de aprendizaje mientras me familiarizo con los fundamentos de Python.

> Hecho por [@adrv6](https://github.com/adrv6)

---

## 📂 Estructura
```
python-exercises/
│
├── Conditionals/
│   ├── coin_flip.py        # if/else con random
│   ├── grades.py           # condicional simple con input
│   ├── magic8.py           # múltiples ramas elif
│   ├── ph_levels.py        # elif encadenados
│   ├── sorting_hat.py      # lógica con múltiples condiciones
│   └── the_cyclone.py      # condiciones combinadas (and)
│
├── Loops/
│   ├── enter_pin.py        # bucle while
│   └── guess.py            # while con múltiples condiciones
│
├── List and functions/
│   ├── fortune.py          # funciones + random
│   ├── inventory.py        # listas, min/max
│   ├── menu.py             # funciones con parámetros
│   ├── prices.py           # funciones + listas + bucles
│   ├── reading_list.py     # manipulación de listas
│   └── anime_list.py       # operaciones con listas (count, index, pop, set)
│
├── Codewars/
│   └── likes.py            # "¿A quién le gusta esto?" - condicionales + f-strings
│
└── README.md
```

## 🧠 Temas cubiertos

- Condicionales (`if`, `elif`, `else`)
- Bucles (`while`)
- Listas y métodos de lista
- Funciones y parámetros
- Módulo `random`
- Entrada de datos con `input()`
- Formato de cadenas con f-strings

## ▶️ Cómo ejecutar

Asegúrate de tener Python 3 instalado. Luego ejecuta cualquier archivo así:
```bash
python coin_flip.py
```

Algunos ejercicios requieren entrada del usuario — simplemente sigue las instrucciones en la terminal.

---

## 📌 Notas

Estos son ejercicios de aprendizaje, no código de producción. Algunos archivos son intencionalmente simples — el objetivo era entender el concepto, no escribir la solución más optimizada.
